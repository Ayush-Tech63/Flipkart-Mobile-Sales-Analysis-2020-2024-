# Flipkart-Mobile-Sales-Analysis-2020-2024-
📊 Flipkart Mobile Sales Analysis (2020–2024) 🔷 Project Overview  This project is an interactive Excel Dashboard built to analyze mobile sales data from 2020 to 2024.  The dashboard provides insights into sales performance, model popularity, discounts, and brand trends using Pivot Tables, Charts, and Slicers.

# 🎯 Objective

Analyze mobile sales performance

Identify top-performing models and colors

Track yearly and monthly trends

Study discount impact on sales

Build a professional interactive dashboard

# 🛠 Tools & Skills Used

Microsoft Excel

Pivot Tables

Pivot Charts

Slicers

Data Cleaning

Conditional Formatting

Dashboard Design

Business Data Analysis
